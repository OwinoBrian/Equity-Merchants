Client Workflow

How the client should use it

1. Add a listing

Click Open Add Listing Form, fill in the property details, upload all photos, and set Status to Active when ready to publish.

2. Edit a listing

Click Open Listings Editor, search for the property, then update price, description, photos, or status. Website changes appear after the data refreshes from Airtable.

3. Hide a listing

Instead of deleting immediately, change Status from Active to something like Inactive or Sold. That removes it from the website while keeping the record.

4. Delete permanently

If they truly want a property gone, they can delete the row directly in Airtable from the editor view. It will disappear from the website the next time the site loads listings.
