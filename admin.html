How To Use It

Follow these steps whenever you update properties.

1. Add a new listing

Click Open Add Listing Form, fill in the property details, upload the photos, and set Status to Active when you want it published.

2. Edit an existing listing

Click Open Listings Editor, find the property, then update the description, price, location, type, status, or photos as needed.

3. Hide a listing from the website

Change the listing Status from Active to another value such as Inactive or Sold. This removes it from the live website without deleting it.

4. Delete a listing permanently

If a listing should be removed completely, delete that row directly in Airtable from the editor view.
